# 美股王直播平台 — 需求与术语

## 1. 业务背景

- **美股王**：十年美股培训教育品牌，提供付费看盘软件（约 200 美元/年）与培训/会员服务。
- **盈立证券**：与美股王相关的券商之一；直播场景中主要涉及的是**盈透证券 (Interactive Brokers)** 交易界面。
- **直播场景**：交易会员/主播在本地同时使用「美股王看盘」与「盈透下单」，通过本平台将两套界面（及美股王多窗口）实时同步给观众。

## 2. 功能需求

### 2.1 信源

| 信源 | 用途 | 窗口数量 |
|------|------|----------|
| 美股王交易软件 | 看盘、技术分析、多品种行情 | 可多窗口（多屏行情） |
| 盈透证券 (IB) | 下单、持仓、账户 | 通常 1 个主界面 |

### 2.2 直播形式

- **双源同播**：同一场直播中同时展示「美股王界面」与「盈透证券界面」。
- **多屏行情**：若主播在美股王中打开多个前端行情窗口，直播画面应展示所有这些窗口（或可配置的子集），而非仅一个窗口。
- **实时同步**：界面内容变化应低延迟呈现在直播流中。

### 2.3 多平台同步推流

- **一次开播，多平台分发**：合成后的直播流同时推送到多个平台。
- 已规划支持的平台：
  - YouTube Live
  - TikTok Live
  - Twitch
  - Bilibili（哔哩哔哩）
  - 自建直播平台
  - 任意自定义 RTMP 地址
- 主播在 app 中配置各平台的 Stream Key / 推流地址，按需启停各路推流。
- 各平台推流状态实时可见（连接中 / 推流中 / 断开 / 错误）。

### 2.4 主播侧

- 打开美股王软件、盈透证券后，通过本直播软件：
  - 选择要采集的窗口（美股王各窗口、盈透窗口）。
  - 预览合成后的画面（多屏布局）。
  - 配置多平台推流目标。
  - 一键开始/停止直播（同时推送到所有已启用的平台）。

### 2.5 观众侧

- 观看端看到的是「合成后的单路直播」：包含美股王多窗口 + 盈透界面，布局由主播端配置或使用默认布局。
- 观众可在 YouTube / TikTok / Bilibili 等任意已推流的平台观看。

## 3. 非功能需求（待细化）

- **延迟**：尽量低延迟（具体目标待定；若用 WebRTC 可优于 RTMP）。
- **清晰度**：支持分辨率/码率配置，保证行情数字与图表可读。
- **平台**：优先支持主播常用环境（如 Windows/macOS）。
- **隐私与安全**：不采集与交易无关的桌面区域；不存储账户密码等敏感信息。

## 4. 术语表

| 术语 | 含义 |
|------|------|
| 美股王软件 | 美股王提供的付费看盘/分析软件，支持多前端行情窗口 |
| 盈透 / IB | Interactive Brokers，交易执行券商 |
| 多屏行情 | 美股王多个行情窗口在直播中同时展示的形态 |
| 双源同播 | 同一直播流中同时包含美股王界面与盈透界面 |
| 多平台同步推流 / Simulcast | 同一路直播流同时推送到多个直播平台（YouTube、TikTok 等） |
| Stream Key | 各直播平台分配给主播的推流密钥 |
| RTMP | Real-Time Messaging Protocol，主流直播推流协议 |
